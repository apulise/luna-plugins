# Luna Plugins by apulise

A collection of plugins for **[Tidal Luna](https://github.com/Inrixia/TidaLuna)** 

## Plugins

### @apulise/api

Based of vmohammad's api plugin. -> https://github.com/vMohammad24/luna-plugins/plugins/api
Provides a local HTTP and WebSocket API for real-time access to Tidal Data. The API server runs on port **24123**.

For more information please visit it's documentation [here](https://github.com/apulise/luna-plugins/tree/master/plugins/api).
